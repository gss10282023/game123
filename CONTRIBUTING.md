# Contributing

Thanks for your interest in contributing to PacmanFX!

## Development setup

- JDK 17+
- No global Gradle required (use the Gradle Wrapper)

### Run locally

```bash
./gradlew run
```

### Run tests

```bash
./gradlew test
```

### Build distribution

```bash
./gradlew installDist
```

## Submitting changes

- Prefer small, focused pull requests.
- Run `./gradlew build` before opening a PR.
- Include a short description of what changed and why.

## Code style

- Keep changes minimal and consistent with the existing style.
- Avoid reformatting unrelated code.
